HTML

HTML stands for Hyper Text Markup Language. HTML is the standerd markup language for creating web page. HTML is used for defining layout of a page. There are many tags in HTML to make a web page layout. A tag is like a container for either content or other HTML tags. Most of the HTML element have opening & closing tag with content in between opening and closing tag. Thre are some tags have no content these are called empty elements.

CSS

CSS stands for Cascading Style Sheet.CSS is a stylesheet language used to describe the presentation of a document written in HTML.CSS describes how element should be rendered on screen, on paper, in speech, or on other media. In CSS there are selectors that we use to select the HTML code and then we design our web page. It allows you to apply styles to web page.CSS describes how a webpage should look: it prescribes colors, font, spacing, and much more.In short, you can make your websitenlook however you want. CSS lets devlopers and designer define how it behaves, including how elements ars positioned in the browser.
